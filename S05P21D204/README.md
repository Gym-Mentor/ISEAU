# SSAFY Bigdata project

### Our default branch is develop branch.


## How to Run

### ISEAU

```sh
cd backend
pip install -r requirements.txt
python parse.py
python analyze.py
python visualize.py
```


**Backend**

```sh
cd ISEAU/backend
pip install -r requirements.txt
python manage.py runserver
```

**Frontend**

```sh
cd ISEAU/frontend
yarn install
yarn dev
```
